# Contributing

Thanks for your interest in improving Argument Atlas IELTS.

## Ways to contribute

- Improve prompt handling for additional IELTS Task 2 question types
- Add better topic libraries for common IELTS themes
- Improve UX copy and layout clarity
- Suggest more realistic example directions and off-topic traps

## Before opening a pull request

1. Keep changes focused and easy to review
2. Explain what user problem your change improves
3. If you change the planning logic, include one or two sample prompts in the PR description

## Scope guideline

This project is intentionally narrow. Contributions that make the Task 2
planning experience clearer are a better fit than turning the repo into a full
IELTS learning platform.
